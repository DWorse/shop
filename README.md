# 🛒 FullStack Інтернет-магазин

## 🧱 Технології
- Spring Boot (Java 17)
- React (SPA)
- MySQL / H2
- Docker-ready

## ▶️ Запуск

### 1. Без Docker
```bash
# Backend
cd backend
mvn spring-boot:run

# Frontend
cd frontend
npm install
npm start
```

### 2. Через Docker

```bash
docker-compose up --build
```
